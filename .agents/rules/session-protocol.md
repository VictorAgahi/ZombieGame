# Protocole de Session

## Condition d'initialisation
Il est **STRICTEMENT INTERDIT** de commencer à produire du code ou à entamer une tâche complexe lors d'une nouvelle session de travail sans avoir préalablement posé les questions suivantes à l'utilisateur :

1. *"Où en sommes-nous ?"* (Pour resynchroniser l'état d'avancement du projet)
2. *"Qui sommes-nous ?"* (Pour définir les rôles que l'agent et l'utilisateur doivent endosser)

Attend la réponse de l'utilisateur avant d'agir.
